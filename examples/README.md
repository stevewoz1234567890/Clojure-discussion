# Examples

Small **Clojure CLI** projects you can run with **`clojure -M -m …`** from each directory (requires [Clojure CLI](https://clojure.org/guides/install_clojure)).

| Example | Topic |
|--------|--------|
| [clojisr-docstrings](clojisr-docstrings/) | ClojisR 1.1.0 — R help as var metadata (`:docstrings? true`) |
| [reify-meta](reify-meta/) | Clojure **1.12.5-alpha1** — `reify` and reader metadata ([CLJ-2945](https://clojure.atlassian.net/browse/CLJ-2945)) |
| [javadbf-roundtrip](javadbf-roundtrip/) | **javadbf** — `:import` `com.linuxense.javadbf`, write/read a `.dbf` |
| [k7-hello](k7-hello/) | **k7** — disk queue enqueue / poll / ack |
| [eido-circle](eido-circle/) | **Eido** — declarative circle → `target/eido-circle.png` |

Related notes live under [`../discussions/`](../discussions/).
