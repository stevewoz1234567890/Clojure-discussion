# GraalVM — downloads site, Community editions, and Oracle GraalVM

A short **[Clojurians](https://clojurians.slack.com)** thread asked whether everyone was simply using builds from **[graalvm.org/downloads](https://www.graalvm.org/downloads/)**, with the observation that **Community** GraalVM editions appeared to have stopped being published—similar questions were raised **in GraalVM’s Slack** without satisfying answers **at the time**.

This note does **not** summarize Oracle’s roadmap; treat it as **reported context** from that exchange.
